How it works

1

Add your repos

Point Agent Workspace at your GitHub folder. It auto-detects your projects, branches, and worktrees. Supports any repo structure — monorepos, multi-repo setups, or solo projects.

2

Launch agents

Start Claude, Codex, or any CLI agent in any worktree with one click. Each agent gets its own terminal with full shell access. Run as many as your plan allows — in parallel.

3

Review & ship

When agents finish, review their PRs with the built-in diff viewer. Merge or re-prompt — without leaving the workspace. The tier system lets you prioritize what to review first.

Bring Your Own Plan

Already paying for Claude Max or Codex? Use those same plans here — Agent Workspace just wraps your CLI tools. No extra subscription, no API key middleman. Log in once, run everywhere.

100% Local

Everything runs on your machine. No cloud relay, no usage tracking, no phone-home analytics. Your code stays on your hardware. Period.

🔒

Real Terminals

Each agent runs in a real PTY session — not a sandboxed wrapper. Your CLI tools behave exactly like they do in your own terminal. When Claude Code or Codex updates, nothing breaks.

⚡

Workspaces & Worktrees

Group your repos into workspaces. Each repo gets multiple git worktrees so agents can work on different branches simultaneously without stepping on each other.

Built-in Code Review

See every PR your agents create with file tree, side-by-side diff, and commit history. Merge or send back with feedback — all without leaving the workspace.

Commander

A dedicated AI terminal that can see and control all your other agents. Tell Commander to "launch 5 Codexes on my Trello backlog" and it will create worktrees, start agents, send prompts, and track progress — all automatically.

Task Integration

Connect your Trello board and launch agents directly from task cards. Cards become prompts — agents work the ticket, create PRs, and you review. GitHub Issues and Linear coming soon.

✅

Tier System

Tag agents by priority so you know what needs attention. T1 Focus — you're actively watching. T2 Review — ready for your eyes. T3 Background — batch-prompted, check later. T4 Overnight — let it cook. Filter the view to see only what matters right now.

Built by Daily Use

This isn't a demo project. Agent Workspace has been used every day since July 2025 to ship real production code across multiple games, tools, and websites — including building itself. 9 months of daily refinement.
