Bring Your Own Plan

Works with your Claude Max subscription, Codex plan, or your own API keys. Use your existing CLI logins — your credentials never leave your machine.

Runs Locally

Runs on your own hardware. Access through the desktop app or the browser. Zero external telemetry, zero data collection.

🔒

Native Terminals

Uses real terminal sessions under the hood — not a wrapper, not an emulator. CLI tools run exactly as they would in your own shell. Low chance of breakage across versions.

⚡

Workspaces & Worktrees

Group git repositories into workspaces. Each workspace supports multiple worktrees running side by side — different branches, different repos, same view.

Built-in Diff Viewer

Quick access to pull requests with a full inbuilt code review tool. Review diffs without leaving your workspace.

Commander

A top-level AI (usually Claude, but configurable) that controls the interface. It can send commands to any terminal, launch agents, pull tasks from your task board — hands-free automation from a single prompt.

Task Integration

Pull tasks from Trello today. GitHub Issues, Linear, and locally-backed tasks coming soon.

✅

Tier System

Organize sessions by priority. T1 Focus for active deep work, T2 Backup runs alongside or when T1 is blocked, T3 Background for batch prompt and batch review, T4 Overnight for long-running autonomous tasks. Hide and show as needed.

Shipped With Agent Workspace

Built in July 2025 and used every day since. Refined over 8 months of daily use — 100% of our code has been shipped using Agent Workspace.
