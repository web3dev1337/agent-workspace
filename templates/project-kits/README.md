# Project Kits

This directory provides stable template kit paths for project scaffolding.

Current kits mirror the starter content under `templates/scaffolds/`:

- `website-starter`
- `hytopia-game-starter`
- `node-typescript-tool`
- `generic-empty`

`scripts/create-project.js` supports both `templates/project-kits/*` and `templates/scaffolds/*` paths for backward compatibility.
